# cart
simple store cart

## How to use 
Deployed on Heroku: https://cart-actum.herokuapp.com/

To run in development mode:
```
yarn install
yarn start:dev
```
and open http://localhost:3000/

To build and run on production:
```
yarn build
yarn start
```

To run test cases:
```
yarn test
yarn test:coverage
```

## Used technologies
- React, Redux, redux-actions, reselect
- Typescript
- Jest, Enzyme
- Babel
- Eslint
- Webpack
